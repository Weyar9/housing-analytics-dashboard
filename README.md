# Housing Analytics Dashboard

## Project Overview

This project analyzes Canadian housing market data using Python, SQL, and data visualization techniques.

The objective is to explore housing trends, compare prices across cities, and generate insights that support data-driven decision making.

This project demonstrates skills in:

* Data Analytics
* Data Cleaning
* Data Validation
* Exploratory Data Analysis (EDA)
* Data Visualization
* Business Reporting
* SQL Analytics
